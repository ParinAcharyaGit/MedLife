# MedLife
A drug shelf-life tracker for pharmacies
Targeting UN SDG3: Save more lives, save money, reduce wastage.

Core features
- Input name, date of purchase, date of manufacture, expiry date, batch number, serial number, shelf location?, company sourced from, quantity, price
- Frontend Functions: update stock, delete item, auto/custom reminders based on batch/exp date, stock tracking, expiry heatmap
- Database for data storage, item in/out records and basic user authentication (username, password)

Additional features
- Donate/Transfer drugs to nearby clinic facing shortage (Improves access to drugs in underserved clinics)
- AI-powered forecasting for re-stocking based on consumption trends
